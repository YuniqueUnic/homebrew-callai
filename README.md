# homebrew-callai

Homebrew tap for [callai](https://github.com/YuniqueUnic/callai).

## Install

```bash
brew tap YuniqueUnic/callai
brew install --cask callai-app   # desktop GUI
brew install callai             # CLI / daemon
```

## Notes

- Desktop builds may not be Apple-notarized; see caveats after cask install.
- Manifests are generated from GitHub Releases in the main callai repo (`packaging/`).
